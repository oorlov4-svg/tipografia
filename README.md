📌 Polygraphy — Web Project

Polygraphy is a lightweight static website built using pure HTML and CSS. The project focuses on clean structure, semantic markup, and responsive layout without using any JavaScript or external frameworks.

🚀 Features
🧱 Semantic and well-structured HTML5 markup
🎨 Modern styling with CSS3
📱 Responsive layout for different screen sizes
⚡ Fast loading with no dependencies
🧼 Clean and readable codebase
🛠️ Technologies Used
HTML5
CSS3
🎯 Purpose

This project was created as a practice to improve core frontend skills and demonstrate the ability to build simple, responsive web pages using only basic web technologies.
